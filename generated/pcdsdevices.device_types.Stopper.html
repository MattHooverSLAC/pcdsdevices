pcdsdevices.device_types.Stopper¶

class pcdsdevices.device_types.Stopper(prefix, *, name, **kwargs)¶

Controls Stopper

A base class for a device with two limits switches controlled via an external command PV. This full encompasses the controls Stopper installations as well as un-interlocked GateValves

Parameters:
  • prefix (str) – Full PPS Stopper PV
  • name (str) – Alias for the stopper
commands¶

Enum – An enum with integer values for open_valve, close_valve values
